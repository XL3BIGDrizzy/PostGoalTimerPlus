# PostGoalTimerPlus

PostGoalTimerPlus lets you disable the post-score countdown in Super Battle Golf, or extend it beyond the vanilla 60 second limit.

Added a config for the post-score countdown.

The default value is "Disabled", which disables the timer altogether.

## Configuration

- Disabled
- 1 minute
- 2 minutes
- 3 minutes
- 4 minutes
- 5 minutes

## Installation

Install through Thunderstore Mod Manager, or manually place `PostGoalTimerPlus.dll` inside your BepInEx plugins folder.

I have had reports that some anti virus software have flagged this as malicious. This is a false positive.
